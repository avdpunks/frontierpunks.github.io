If you've been following our blog for a while, you know where we started: Azure Virtual Desktop. Deployment scripts, image templates, session host quirks, the odd rant about SAS tokens. AVDPunks was born out of that world, and honestly, we loved every minute of it.

But here's the thing — the world we've been writing about has outgrown the name.

## Why a new site?

Over the last couple of years, our posts stopped being "just AVD." We were writing about Windows 365, Dev Box, Azure networking changes, Intune, Autopilot, Azure Image Builder, Windows Insider builds — basically anything sitting on the edge of the Microsoft cloud and end-user computing stack. AVD was never really the whole story. It was just where we started digging.

"AVDPunks" made that hard to see. Meanwhile the interesting stuff — the stuff we actually get excited to write about — was happening at the frontier: new Azure platform changes, Windows 365 evolving fast, and now AI agents showing up in the exact same corner of the stack we've always lived in. Windows 365 for Agents, Computer-Using Agents (CUA), agents that need a desktop to actually get work done — this is squarely our territory, and it's moving fast.

So instead of trying to cram everything under one name, we built a brand new home for it.

## Welcome to FrontierPunks

Still the same two dudes behind the keyboard — Ben and Daniel. Same irreverent, no-fluff, "let's just get this working" attitude. Same commitment to sharing real scripts, real fixes, and real lessons learned instead of marketing copy. FrontierPunks is a new site, built from scratch, for everything sitting at the frontier.

Expect posts on:

- **AI agents** — Windows 365 for Agents, Computer-Using Agents (CUA), and everything about giving agents a desktop to actually work in
- **Whatever's next** — new Azure services, AI-assisted IT ops, and anything else sitting at the frontier of enterprise cloud computing that's worth digging into
- **Azure Virtual Desktop & Windows 365** — still core to what we do
- **Networking and platform changes** that quietly break your environment if nobody warns you (we'll keep warning you)

## What about avdpunks.com?

**avdpunks.com isn't going anywhere and isn't changing.** Every existing post, every deep link, every bookmark keeps working exactly like before — same site, same name, same archive of AVD and Windows 365 deep-dives.

FrontierPunks is a new, separate site sitting alongside it — the place where we'll cover new ground. Two sites, same two dudes, one mission: figure out the stuff the documentation doesn't cover yet, break it in a lab first, and hand you the working answer.

Thanks for reading — see you out there on the frontier.

— Ben & Daniel
